# 🌟 Udyog Saarthi: Empowering Inclusive Careers

**Udyog Saarthi** is a Progressive Web Application (PWA) dedicated to bridging the gap between talent and opportunity for adults undergoing job coaching, specifically supporting those under the **4% reservation scheme** in NIEPMD and other institutions.

---

### 🌐 Live Deployment
*   **Live Website**: [https://udyog-saarthi.vercel.app](https://udyog-saarthi.vercel.app)
*   **API Backend**: [https://udyog-saarthi.onrender.com](https://udyog-saarthi.onrender.com)

---

## 🚀 The Vision
Adults undergoing job coaching often face challenges in finding a centralized platform that speaks their language—one that provides not just job listings, but the **tools, training, and confidence** needed to thrive in a professional environment. 

**Udyog Saarthi** (The Professional Guide) is designed to be that companion, offering a high-contrast, simple, and accessible interface that empowers users to take charge of their career journey.

## ✨ Key Features

### 🛠️ For Job Seekers
*   **AI-Powered Resume Builder:** Create professional, single-column resumes in minutes. Upload your photo and download a clean PDF ready for any employer.
*   **Interactive Interview Simulator:** Practice common interview questions using your voice or text. Get instant feedback on your confidence and clarity.
*   **Accessible Job Board:** Find job postings specifically tagged for reservation categories. Use the "Apply with Assistant" feature to guide you through the process.
*   **Guided Learning:** Access a centralized hub of training modules (videos, articles, and quizzes) to sharpen your skills.
*   **High-Contrast & Simple Modes:** A UI designed for everyone, with specialized modes to assist users with visual or cognitive impairments.

### 🏢 For Employers
*   **Inclusive Hiring:** Post jobs specifically for the PWD reservation scheme.
*   **Applicant Management:** Easily manage and review applications from a dedicated dashboard.

### 🎓 For Trainers & Coaches
*   **Resource Sharing:** Create and publish training modules to guide your students.
*   **Engagement Tracking:** Monitor how students are interacting with your materials.

## 💻 Tech Stack
*   **Frontend:** React.js, Tailwind CSS, Lucide Icons.
*   **Backend:** Laravel 11 (PHP 8.2), Docker.
*   **Database:** MongoDB Atlas (Cloud NoSQL).
*   **Authentication:** Laravel Sanctum (Custom MongoDB Bridge).
*   **Hosting:** Render (Backend), Vercel (Frontend).
*   **PDF Generation:** html2pdf.js.
*   **Speech Services:** Web Speech API (Recognition & Synthesis).

## 🛠️ Quick Setup

### Backend (Laravel)
1. Navigate to `/backend`
2. Run `composer install`
3. Configure your `.env` file and database.
4. Run `php artisan migrate --seed`
5. Start the server: `php artisan serve`

### Frontend (React)
1. Navigate to `/frontend`
2. Run `npm install`
3. Start the dev server: `npm run dev`
